# BMAD Bootstrap

Ce dossier initialise un workflow BMAD minimal pour ce projet.

## Demarrage rapide

1. Charger la skill `bmad-help` puis demander: `que dois-je faire maintenant ?`
2. Lancer la premiere etape: `bmad-analyst` avec l'objectif de valider `docs/s-019.md`
3. Enchainer avec `bmad-dev-story` pour l'implementation
4. Valider avec `bmad-qa`
5. Clore avec `bmad-tech-writer` pour `docs/findings-report.md`

## Fichiers

- `_bmad/_config/bmad-help.csv`: catalogue des workflows
- `_bmad/core/config.yaml`: configuration module et langue
